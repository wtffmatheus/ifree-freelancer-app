# 🚀 iFree - Plataforma de Freelancers

Sistema para conectar freelancers a restaurantes, permitindo candidatura em vagas e gerenciamento de usuários.

## 🔧 Tecnologias
- React
- Node.js
- Firebase Authentication
- Banco de dados

## 🔐 Funcionalidades
- Cadastro e login de usuários
- Autenticação com Firebase
- Candidatura em vagas
- Gerenciamento de dados
